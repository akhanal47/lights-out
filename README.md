# Lights-Out
-------
Bare bone extension to force dark-mode ON websites

> **⚠️ INFO ⚠️** \
> This probably will not work ON all website

----
### Manual Setup Instructions:
There are two ways to install the extension manually, cloning the repo, downloading the zipped ```.zip``` file from the GitHub Releases page.

To use any of these methods please 
1. Go to ```chrome://extensions/```
2. Turn ON "Developer Mode" (In Google Chrome; it's a toggle ON the top right corner)


#### Cloing the Repo
1. Clone the repo using ```git clone https://github.com/akhanal47/lights-out```
1. Enable the "Developer Mode"
2. Select "Load Unpacked" button & select the folder where the repo is cloned

#### Using the Zip File (<span style="color: Green">Recommended</span>)
1. Enable the "Developer Mode"
2. [Dowload](https://github.com/akhanal47/lights-out/releases) and save latest ```lights-out.zip``` (latest date) file then unzip it to your desired location.
3. Select "Load Unpacked" button & select the folder where you unzipped the extension files.

----
### Usage

Click the Lights Out icon in your browser toolbar to open the extension.

**Enabling dark mode ON a site**\
Dark mode is OFF by default. Visit any site you want darkened, open the extension, and flip the toggle ON. The page will reload in dark mode. It will stay dark every time you visit that site.

**Disabling dark mode ON a site**\
Open the extension ON that site and flip the toggle OFF. The page reloads back to normal.

**Site looks broken after enabling?**\
Some sites may not play well with full dark mode — text may disappear or the whole website may go black. If anything appears to be broken, open the extension and turn ON the **Looks broken?** toggle. This switches to a lighter dark mode that should be easier ON complex sites. You can turn it back OFF anytime if the site gets fixed or you want full dark mode regardless.
